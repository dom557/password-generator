# Password Generator

## Description
The Password Generator is a command-line application built in Python. It generates strong and secure passwords based on user-defined criteria such as length and character types.

## Features
- Generate random passwords with specified length.
- Customize password composition with options for including uppercase letters, lowercase letters, digits, and special characters.
- Simple and intuitive command-line interface.

## Requirements
- Python 3.x

## Installation
1. Clone this repository:
    ```bash
    git clone https://github.com/dom557/password-generator.git
    ```

2. Navigate to the project directory:
    ```bash
    cd password-generator
    ```

## Usage
Run the script `main.py` and follow the on-screen prompts to generate a password according to your preferences.

```bash
python main.py
```

You will be prompted to specify the following:
- Password length (minimum 6 characters)
- Character types to include: uppercase letters, lowercase letters, digits, and special characters

After entering your preferences, the generated password will be displayed on the console.

## Examples
- Generate a 12-character password with uppercase letters, lowercase letters, and digits:
    ```bash
    python password_generator.py
    ```
- Generate an 8-character password with lowercase letters and special characters:
    ```bash
    python password_generator.py
    ```
